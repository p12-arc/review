# ReviewAI - Review Analytics Dashboard
Live deployment: https://p12-arc.github.io/review/

ReviewAI is a modern, responsive dashboard for ecommerce teams to monitor review sentiment, spot emerging issues, and track product performance across marketplaces.

## Highlights

- Sentiment breakdown with trends and priority tagging
- Product-level insights with ratings and key issues
- Alerting panels for spikes and high-risk reviews
- Interactive charts powered by Chart.js
- Responsive, animation-rich UI with clean typography

## Tech Stack

- HTML5, CSS3, JavaScript (ES6)
- Chart.js for visualizations
- No build tooling required

## Pages

- Dashboard: index.html
- Products: products.html
- Future improvements: roadmap.html
- Mock data reference: data.html

## Run Locally

1. Clone the repository
```
git clone https://github.com/p12-arc/review.git
cd review
```

2. Open in a browser
```
open index.html
```

Optional: serve via a local server
```
python -m http.server 8000
```
